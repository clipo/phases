# CMV/LMV macro transect: boundary or smooth cline?

133 decorated assemblages (>= 10 decorated sherds): CMV 20, LMV 113. Decorated types harmonized (analyses 27).
Latitudinal gap between the southernmost CMV and northernmost LMV assemblage in this set: ~48 km.

## Distance-decay across the transect
- Mantel r (decorated BR similarity vs geographic distance) = -0.555 (p = 0.0002).

## Boundary excess (CMV vs LMV, distance-controlled)
- raw within-minus-between region BR = +64.9.
- distance-controlled boundary excess = +51.6 BR units.
  >0 beyond IBD = a step in decorated similarity at the boundary.

## Spatial-drift null on the combined geography (60 sims)
- distance-decay r: observed -0.555, null 95% [-0.604, -0.506] (inside).
- boundary excess: observed +51.6, null 95% [-0.2, +15.2] (OUTSIDE).
  Note: the drift null randomizes type labels, so it cannot reproduce the mutually-exclusive distinctive repertoires; it tests only whether the PROPORTIONAL step exceeds neutral spatial drift.

## CRITICAL CAVEAT
The CMV/LMV decorated difference is confounded with time (early-middle CMV
types vs late LMV types; analyses 27). These spatial statistics describe
PATTERN; they do not separate a synchronic spatial boundary from early-CMV/
late-LMV succession. A synchronic test needs chronological control per
assemblage (independent CMV radiocarbon).
